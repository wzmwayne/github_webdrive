# files.inf 文件格式说明

`files.inf` 是一个纯文本文件，用于存储云盘中所有文件和目录的信息。网页会读取此文件来展示文件列表。

## 文件格式

每行代表一个文件或目录，使用 `|` 符号分隔不同字段：

```
相对路径|显示名称|类型|大小(字节)|更新时间(ISO格式)
```

## 字段说明

1. **相对路径**：文件相对于仓库根目录的路径
2. **显示名称**：在网页上显示的文件/目录名称
3. **类型**：`dir` 表示目录，`file` 表示文件
4. **大小**：文件大小（字节），目录可填 `0`
5. **更新时间**：ISO 8601 格式的时间戳（可选）

## 示例

```
# 注释行以 # 开头，会被忽略
docs/|文档|dir|0|2026-01-30T10:00:00Z
docs/readme.md|README.md|file|1024|2026-01-30T10:00:00Z
images/|图片|dir|0|2026-01-30T10:00:00Z
images/logo.png|logo.png|file|524288|2026-01-30T10:00:00Z
project.zip|项目压缩包|file|1048576|2026-01-30T10:00:00Z
```

## 注意事项

- 路径使用正斜杠 `/` 作为分隔符
- 目录路径应以 `/` 结尾
- 空行和以 `#` 开头的行会被忽略
- 文件应放在仓库的 `files/` 目录下或其他合适位置
- 更新此文件后，网页会自动显示最新的文件列表
